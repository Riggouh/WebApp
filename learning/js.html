Ein Hauch von JavaScript

JavaScript bringt Interaktivität auf deine Webseite. Du kannst auf Klicks reagieren, Inhalte dynamisch ändern, Formulare prüfen oder Animationen steuern.

Beispiel für JavaScript 
document.querySelector("button").addEventListener("click", function() {
  alert("Hallo Welt!");
});

Wichtige JavaScript-Grundlagen

Befehl / Objekt Beschreibung Beispiel
document.querySelector() Wählt das erste Element, das zum CSS-Selektor passt document.querySelector("p")
addEventListener() Fügt einem Element einen Ereignis-Listener hinzu (z.B. Klick) button.addEventListener("click", callback)
alert() Zeigt eine Popup-Nachricht alert("Hallo!");
console.log() Gibt Text oder Werte in der Browser-Konsole aus (für Debugging) console.log("Test");
let / const Variablen-Deklaration (let änderbar, const konstant) let name = "Max";
const pi = 3.14;
function Definiert eine Funktion function sagHallo() { alert("Hallo!"); }
if / else Bedingte Anweisungen if (x > 5) { ... } else { ... }
for Schleife for (let i=0; i<5; i++) { ... }

Ein erweitertes Beispiel

Klick-Event mit Textänderung 
const button = document.querySelector("button");
const message = document.createElement("p");
message.textContent = "Noch kein Klick.";
document.body.appendChild(message);

button.addEventListener("click", () => {
  message.textContent = "Du hast auf den Button geklickt!";
  console.log("Button wurde geklickt.");
});
