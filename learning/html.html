HTML Grundlagen

HTML (HyperText Markup Language) ist die Struktur deiner Webseite. Du baust damit Überschriften, Absätze, Listen, Bilder, Links und vieles mehr.

Beispiel für HTML 
<h1>Hallo Welt</h1>
<p>Das ist ein Absatz.</p>
<ul>
  <li>Listelement 1</li>
  <li>Listelement 2</li>
</ul>
<a href="https://example.com">Link zu Example.com</a>

Wichtige HTML-Tags auf einen Blick

Tag Beschreibung Beispiel
<h1> – <h6> Überschriften in verschiedenen Größen <h2>Meine Überschrift</h2>
<p> Absatz <p>Dies ist ein Absatz.</p>
<a> Hyperlink zu einer anderen Seite oder Ressource <a href="https://google.de">Google</a>
<img> Bild einfügen (benötigt src-Attribut) <img src="bild.jpg" alt="Beschreibung">
<ul> / <ol> Ungeordnete (Punkte) und geordnete (Nummerierte) Listen <ul><li>Item 1</li></ul>
<ol><li>Erster</li></ol>
<div> Container für andere Elemente, um sie zu gruppieren <div class="container">Inhalt</div>
<span> Inline-Container für Text oder kleine Elemente <span style="color:red;">Rot gefärbter Text</span>

Weitere wichtige HTML Konzepte

  • Attribute: Zusätzliche Informationen zu einem Tag, z.B. href bei Links oder src bei Bildern.
  • Verschachtelung: HTML-Elemente können ineinander verschachtelt werden, z.B. eine Liste innerhalb eines Absatzes.
  • Semantik: HTML-Tags haben eine Bedeutung, z.B. <header> für den Kopfbereich, <footer> für den Fußbereich.

Hier noch ein erweitertes Beispiel mit mehreren Elementen:

Komplexeres Beispiel

  <h1>Meine Webseite</h1>
</header>

    <li><a href="#contact">Kontakt</a></li>
  </ul>
</nav>

<section id="home">
  <h2>Willkommen!</h2>
  <p>Das ist eine Beispielwebseite.</p>
  <img src="beispielbild.jpg" alt="Beispielbild">
</section>

<footer>
  <p>© 2025 Meine Webseite</p>
</footer>
