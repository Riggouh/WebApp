CSS Grundlagen

CSS (Cascading Style Sheets) verleiht deiner Webseite Stil. Du kannst Farben, Layouts, Schriftarten, Abstände und vieles mehr definieren.
Oft als eigenständige Datei ("style.css").

Beispiel für CSS 
body {
  background-color: #f4f4f4;
  color: #333;
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 20px;
}

Wichtige CSS-Eigenschaften

Eigenschaft Beschreibung Beispiel
color Textfarbe color: red;
background-color Hintergrundfarbe background-color: #00ff00;
font-size Schriftgröße font-size: 16px;
margin Außenabstand (Abstand zum Nachbar) margin: 10px;
padding Innenabstand (Abstand zum Inhalt) padding: 5px;
border Rahmen border: 1px solid black;
width / height Breite und Höhe eines Elements width: 100px;
height: 50px;
display Bestimmt, wie ein Element dargestellt wird (z.B. block, inline, flex) display: flex;
text-align Textausrichtung text-align: center;

CSS Selektoren

CSS benutzt Selektoren, um Elemente auszuwählen, die gestylt werden sollen.

  • element (z.B. p) – alle Elemente dieses Typs
  • .class – alle Elemente mit der Klasse
  • #id – das Element mit der ID
  • element.class – Elemente eines Typs mit einer bestimmten Klasse

Beispiele:

p {
  color: blue;
}

.highlight {
  background-color: yellow;
}

#header {
  font-weight: bold;
}
